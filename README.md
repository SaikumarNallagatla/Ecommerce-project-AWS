# AWS E-Commerce Platform on AWS

## Project Overview

This project demonstrates a production-style cloud-native e-commerce application deployed using modern DevOps practices and AWS cloud services.

The goal of this project is to showcase Infrastructure as Code (IaC), containerization, CI/CD automation, Kubernetes orchestration, monitoring, and cloud deployment workflows used by DevOps Engineers in real-world environments.
